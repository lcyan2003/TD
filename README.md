# TD：108 課綱科技領域互動教材

本專案用於製作 108 課綱科技領域（Technology Domain）相關教材，目標是以網頁形式發布，讓學生能透過公開連結直接進行互動學習。

## 教材目標

- 支援國中小科技領域課程的課堂教學與自學活動。
- 整合理論概念、生活情境、互動任務與學習檢核。
- 便於後續部署至 GitHub Pages 或其他靜態網頁平台。

## 初始結構

- `index.html`：網站首頁與學習入口。
- `styles.css`：版面與視覺樣式。
- `script.js`：互動任務與前端行為。
- `AGENTS.md`：專案協作與回覆偏好。

## 後續建議

1. 依 108 課綱科技領域核心素養與學習內容建立教材單元。
2. 將課程分成「資訊科技」與「生活科技」兩大主軸。
3. 逐步加入互動模擬、形成性評量、學習單與教師備課資源。
4. 啟用 GitHub Pages，作為學生可直接存取的公開教材網站。
